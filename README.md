RU: # Загрузчик изображений из Excel
Это готовый exe-загрузчик для удобной работы с большими массивами и базами данных фотографий или изображений. Программа создает папку и загружает в нее фотографии, используя указанные ссылки, нумеруя и подписывая их.
Первый столбец может содержать текстовые названия фотографий или коды товаров.
Таким образом, если столбец A1 содержит название «товар», а столбцы B1, C1, D1 содержат ссылки на фотографии, то будет создана папка «товар», и в нее будут загружены фотографии «товар_1», «товар_2», «товар_3».

Вы можете скачать исполняемый файл релиза на странице релизов или по ссылке: https://github.com/ThreePerCento/Photo_downloader_from_links_to_Excel/releases

EN: # Excel photo uploader
This is a ready-made exe downloader for convenient work with large arrays and databases of photographs or pictures. The program creates a folder and downloads photographs into it using the specified links, numbering and signing it.
The first column can contain text names of photographs or product codes.
Thus, if column A1 contains the name "item", and columns B1, C1, D1 contain links to photographs, then the folder "item" will be created and the photos "item_1", ​​"item_2", "item_3" will be loaded into it.
You can download the release exe on the releases page or by following the link: https://github.com/ThreePerCento/Photo_downloader_from_links_to_Excel/releases
